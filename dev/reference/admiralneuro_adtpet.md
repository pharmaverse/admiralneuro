# Tau PET Scan Analysis Dataset - Neuro

An updated ADaM ADTPET dataset using NV, AG, SUPPNV, ADSL

## Usage

``` r
admiralneuro_adtpet
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 34
rows and 47 columns.
