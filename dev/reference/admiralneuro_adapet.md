# Amyloid PET Scan Analysis Dataset - Neuro

An updated ADaM ADAPET dataset using NV, AG, SUPPNV, ADSL

## Usage

``` r
admiralneuro_adapet
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 83
rows and 50 columns.
